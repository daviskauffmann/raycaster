# Raycaster

## Getting Started

### Download

```sh
git clone https://github.com/daviskauffmann/raycaster.git
cd raycaster
```

### Install Dependencies

Install [CMake](https://cmake.org/).
Install [vcpkg](https://vcpkg.io/).

```sh
vcpkg install sdl2 sdl2-image sdl2-ttf
```

### Build

```sh
mkdir build
cd build
cmake ..
```
