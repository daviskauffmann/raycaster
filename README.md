# Undermountain

## Getting Started

Make sure these are installed.

```txt
SDL2
SDL2_image
SDL2_ttf
```

Build the project.

```sh
make
```

Build and run.

```sh
make run
```

Cleanup all build files and binaries.

```sh
make clean
```

## Configuration

Copy `config.txt` and rename to `config.local.txt` to make local changes to the configuration. This file is not source controlled.
